# circum [![Build Status](https://travis-ci.org/chantastic/circum.svg?branch=master)](https://travis-ci.org/chantastic/circum)

> An array index for any number.


## Install

```
$ npm install --save circum
```


## Usage

```js
const circum = require('circum');

circum([1,2,3], -2);
//=> 1
```


## API

### circum(arr, index)

#### arr

Type: `array`

#### index

Type: `number`
